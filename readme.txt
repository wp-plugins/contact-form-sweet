Simple Contact Form for your WordPress website.

== Description ==
Contact Form By Sweet is a simple Contact Form Plugin, so easy you just have to  activate the plugin and insert the shortcode [sweetcontact-form form='1'] into a page or a post.
we are going to add some cool feature to this Contact Form plugin real soon.
...

Features:
--------
* Full mobile support! Tested for iPhones, iPads and Android devices
* You can choose where to send the messages - this can be a WordPress user or any email address.
* You can choose the message the user will get after pressing "submit" in the contact form or redirect to another site.
* You can add a field for sending a A copy that will be sent to the email address specified while filling out the contact form.
* Ability to choose between horizontal and vertical Contact Form to be displayed.

== Installation ==

1. Upload the `Contact Form` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in "sweetContact" menu.
4. Create a page or a post and insert the shortcode [sweetcontact-form form='1'] into a page or a post.

== Frequently Asked Questions ==

= Where can I find the settings to adjust the Contact Form plugin work after activation? =

In the WordPress dashboard menu you can find the "sweetContact" menu to the settings page.

= How can I add Contact Form to my website? =

You should put the shortcode [sweetcontact-form form='1'] into a page or a post.
